# USB_Watcher
This project is designed to monitor in real time the appearance of new USB devices in Windows 10.
The project code is written in C++ using the LibUsb 1.0 library in Qt Creator 7.0.0 (based on Qt 6.23 (MSVC 2019, 64 bit)).
Library source link https://github.com/libusb/libusb/releases/tag/v1.0.26
